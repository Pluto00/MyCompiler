# 一个编译器前端实现

## feature

1. 对语言大小写不敏感的词法分析

2. slr 语法分析

3. 可视化界面

4. 支持生成中间代码

## usage

1. `pip install flask`

2. `python app.py`

## debug

`python ./SyntaxAnalyzer/SLR.py` -> `./resource/debug_info.xls`
